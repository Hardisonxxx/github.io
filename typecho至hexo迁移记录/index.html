Typecho至Hexo迁移记录

Denglzz 2023-09-05 13:41:09
Categories： Tags：

0 前言

关于笔记网站从动态迁移至静态的想法已有许久，静态网站的本地保存策略是极佳的，相当于二次备份，趁着这几天闲暇之余完成。静态网站的选择有很多，因为只前看到过一款hexo的主题印入脑海许久，即选之。

  • 选用服务器部署静态网站
  • 服务器已经安装宝塔面板

1 保存typecho数据，将其转换为Hexo可用格式

这里需要用到脚本来完成转换，将脚本保存在服务器任意位置（需要新建文件夹存之）

需要用到数据库名、数据库用户名、数据库密码，数据库地址（localhost）

将下列所需之处替换即可：

脚本： >https://github.com/NewbMiao/typecho2Hexo

完成之后，typecho中的文章即可转换为Hexo可用的markdown格式，下载即可；

2 安装本地Hexo

Hexo需要依赖Node.js 我们需要安装它。为了方便同步，我们使用Git。

安装Nodejs以及git

https://nodejs.org/zh-cn //安装Nodejs

https://git-scm.com/downloads //安装 Git

具体过程不再赘述

安装Hexo

在cmd中输入下面代码，使用淘宝npm镜像来代替下载。以后使用cnmp代替nmp即可

npm install -g cnpm --registry=https://registry.npm.taobao.org  
npm install -g hexo-cli

创建目录

hexo init <folder> // folder为你的博客目录名字，任取
cd <folder> // 进入该目录
npm install // 安装依赖
npm install hexo-server --save //安装服务
npm install --save hexo-deployer-git //安装部署git

至此本地博客的基本搭建就完成了。

云服务器配置

安装Git

yum install -y git
或（不同linux自己选择）
apt-get install git

因为本人有点懒 直接使用root用户上传下载。

宝塔面板上新建网站

在网站目录下建立git仓库

git init --bare 随便起个名字.git  //创建Git仓库
chown root:root -R 随便起个名字.git  //授予Git仓库权限

新建钩子文件post-receive

nano 网站目录/随便起个名字.git/hooks/post-receive

往post-receive粘贴

#!/bin/bash
git --work-tree=网站目录  --git-dir=网站目录/随便起个名字.git checkout -f

保存后修改权限：

chmod +x 网站目录/随便起个名字.git/hooks/post-receive

开始从本地往服务器上传

这里需要一个秘钥，在你的物理机上C: .ssh的文件中找到id_rsa.txt文件

复制到服务器中的~/.ssh/authorized_keys文件下。

然后使用ssh测试一下是否可以不用密码连接。

然后在本地刚创建的Hexo网站下找到_config.yml文件，在里面找到deploy，编辑如下

deploy:
  - type: git
    repository: root@你的ip地址:网站目录/随便起个名字.git
    branch: master

然后通过以下命令即可推送：

hexo clean
hexo g
hexo -d

主题优化

我用到的主题是Wujun大佬的tree：https://github.com/wujun234/hexo-theme-tree

我有显示多行数学公式的需求，用了很多latex插件总觉的不尽人意。最后选用了一种比较好用的配合方案：

hexo-filter-mathjax + hexo-renderer-pandoc

hexo-filter-mathjax 用来渲染数学公式

hexo-renderer-pandoc 是一个渲染器，用来渲染Markdown，经过测试，该渲染器对Mathjax的渲染是最好的。

最后他们的配合是不错的

安装

先把所有其他hexo math有关的npm包卸掉，比方说这些：

cnpm un hexo-math --save
cnpm un hexo-renderer-markdown --save

然后安装hexo-filter-mathjax：

cnpm i hexo-filter-mathjax --save

然后安装pandoc: https://pandoc.org/installing.html

然后安装hexo-renderer-pandoc：

cnpm i hexo-renderer-pandoc --save

在_config.yml中添加如下： （这样所有博客都会自动使用MathJax。然后就可以像在Latex里面一样编辑公式了：）

mathjax:
  every_page: true

启用pandoc：

pandoc:
  pandoc_path: C:\Program Files\Pandoc\pandoc.exe
  args:
    - '--no-highlight'   # 禁止编译器对代码进行高亮操作（会修改代码框的html）
    - '--mathjax'
  extensions:
    - -implicit_figures # 防止在图片后面将方括号里的东西作为caption。
    - +gfm_auto_identifiers+angle_brackets_escapable # Not available in pandoc 1.16
    - +pipe_tables+raw_html+fenced_code_blocks
    - -ascii_identifiers+backtick_code_blocks+autolink_bare_uris
    - +intraword_underscores+strikeout+hard_line_breaks+emoji
    - +shortcut_reference_links

引用自

https://seekstar.github.io/2021/11/16/hexo%E6%94%AF%E6%8C%81latex%E9%A3%8E%E6%A0%BC%E7%9A%84%E5%85%AC%E5%BC%8F%E7%BC%96%E8%BE%91/

代码高亮

tree主题是不带代码高亮的，需要对其进行单独设置。

  • 将自带的highlight和prismjs禁用掉
highlight:
  enable: false
  line_number: true
  # auto_detect: true   #把这个也注释掉，有点bug
  tab_replace: 2
  wrap: 
  hljs: 
prismjs:
  enable: false
  preprocess: true
  line_number: true
  tab_replace: ''

  • 在head.ejs中引入highlightjs和css

      <%- css('https://lf3-cdn-tos.bytecdntp.com/cdn/expire-1-M/highlight.js/11.4.0/styles/tomorrow-night-blue.min.css') %>
  <%- js('https://lf9-cdn-tos.bytecdntp.com/cdn/expire-1-M/highlight.js/11.4.0/highlight.min.js') %>

  • 用一个代码主题，这里选用的是a11y-light主题，其他主题可在highlight官网中的demo中找到，并在https://cdn.bytedance.com/?query=highlight.js&version=11.4.0中找到响应的cdn链接替换

      <%- css('https://lf26-cdn-tos.bytecdntp.com/cdn/expire-1-M/highlight.js/11.4.0/styles/a11y-light.min.css') %>

  • 在script中开启

      <script>
  hljs.highlightAll();
  hljs.initLineNumbersOnLoad();
  </script>

然后部署即可。

代码添加行号

highligh默认是不带行号的，我们需要借助highlightjs-line-numbers.js来操作

同样，在头文件中引入：

    <%- js('https://lf6-cdn-tos.bytecdntp.com/cdn/expire-1-M/highlightjs-line-numbers.js/2.8.0/highlightjs-line-numbers.min.js') %>

通常情况下就可以使用了，但是我用的主题tree中main.css中的table标签将border=1，导致代码会有表格将其框住。我们找到main.css，将其中的table标签中的border改成0，如下所示：

table {
  margin: auto auto 15px;
  width: 100%;
  min-height: .01%;
  overflow-x: auto;
  background: transparent;
  border-collapse: collapse;
  border-spacing: 0;
  text-align: left; 
  border: 0px solid black;    
  /* border=0   */
}
table th {
    font-weight: bold;
    padding: 5px 10px;
    border: 1px solid black;
}
table td {
    padding: 5px 10px; 
    border: 1px solid black;
}
