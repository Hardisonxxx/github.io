第0章 操作中遇到的问题

  1. 在导入三维模型后，会遇到三维模型边角面显示不光滑。这不是三维文件的问题，而是ANSA分辨率的问题。只需要使用TOPO>Auxiliaries>Fine操作将不光滑的边界多次点击之后即可改变其分辨率从而达到显示光滑的效果。

第1章 设置、操作相关

1.1 操作

1.1.1 对模型的操作

旋转：ctrl+左键拖拽 平移：ctrl+鼠标中键拖拽 缩放：ctrl+鼠标中键+左键

1.1.2 各类快捷键

F1：俯视图 F2：左视图 F3：正视图 F4：底视图 F5：后视图 F6：右视图 F7：缩小 F8：放大 F9：屏幕适中 F10：默认视图 F11：进入quality criteria-prssentation parameters窗口 F12：打开database窗口，出现在屏幕左侧 Esc取消 关闭 退出 ctrl+shift 显示元件线条构造

1.2 仅选中当前所看到的一个面

MENUBAR>Feature Selection>Select Front Only

基于当前视角仅选择“前方”实体的标志。

1.3 设置

1.3.1 分辨率设置

在Settings下拉菜单中找到Resolution/Tolerances/Units

模型分辨率 Resolution

Perimeter Length: 表示周长，即导入模型之后默认两个节点的距离，一般默认值为20mm, 将这个数值改小会使得模型分辨率提高，我一般都用2-3mm；如果这个数值很大出现很大的变形，与实际在三维图中的模型相差很大。

Distortion angle：表示扭曲角度，例如会在一些小圆角特征径向加点，角度越大加的越多，如果为零则不加，越小的圆角越明显。这里我一般都选择填0°，因为有时圆角加很多节点会导致网格质量下降。

容差Tolerances

Hot POINT matching distance：表示两个节点最小节点长度值，也可以理解为Mesh>Perimeter>Length中输入的最小值。有的时候为了追求很细的网格，需要在这里设置，但同时使在生成网格的时间加长，电脑不是很好的情况下会死机。

单位Units

Scale Current units convert to ：这个很好理解，设置导入模型的单位，默认的是meter(m)，如果你的模型一般都比较小设置Millimeter(mm)是最佳选择。设置完成之后一定要勾选Apply to models，并且点击Apply Units。

注意：在这个界面下设置完成之后，模型导入之后还是没有改变，是因为在Translator下拉菜单中也有一个Resolution/Tolerances/Units，将其中对应的设置填写一样即可成功。

第2章 Topo操作

  1. Hot Point: 在ANSA中，“Hot Point”通常指的是模型上的一个特定点，这个点可能是几何上的关键点，如两条线或多个面的交汇点。这些点在模型编辑和拓扑优化中很重要，因为它们可能是需要特别关注或处理的地方。
  2. CONS (Constraints): 在拓扑编辑中，“Constraints”指的是对模型中某些部分施加的限制或约束条件。这些约束可以帮助在修改模型时保持某些特定的几何特性或关系，如保持两个面平行或一个点在特定曲线上。
  3. Faces: 在三维建模中，“Faces”指的是构成模型表面的平面或曲面部分。在拓扑编辑中，用户可能需要修改、合并或拆分面以改进模型的几何结构或简化网格生成。
  4. Surfaces: 这与”Faces”类似，通常指模型中的曲面或平面区域。在拓扑编辑中，处理表面可能包括平滑、调整或重新定义表面的边界。
  5. Curves: 在模型中，“Curves”指的是定义模型边界或特征的线条。在拓扑优化中，可能需要编辑曲线来改善模型的质量，比如调整曲线形状或优化其布局。
  6. Point: 这指的是模型中的单个点，可能是几何特征的一部分，如曲线的端点或面的角点。在拓扑编辑中，点的位置和关联关系很重要，因为它们影响到整个模型的结构。
  7. wddddddd
  8. Auxiliaries: “Auxiliaries”通常指辅助工具或额外的元素，它们在模型建立或编辑过程中起到辅助作用，但不一定是最终模型的一部分。这可能包括用于建模的辅助线、平面或其他辅助几何体。

2.0 操作顺序

  1. 法向切换
  2. 全选后进行Topo操作，对无法进行Topo操作的边界进行手动逐一修补。

2.1 Topo中点线面的意义

红色表示边界线，如果有红色的线，代表面是不连续的。single按钮控制其开关。

2.2 Hot Points

“Hot Point”通常指的是模型上的一个特定点，这个点可能是几何上的关键点，如两条线或多个面的交汇点。这些点在模型编辑和拓扑优化中很重要，因为它们可能是需要特别关注或处理的地方。

2.2.1 Insert

插入硬点

This function creates or projects Hot Points by hovering positions on CONS or 3d curves or 1d FE elements.

该函数通过悬停在CONS 或3d曲线或1d FE元素上的位置来创建或投影热点。

2.2.2 Project

投影硬点

该函数以CONS或3D Curves的形式，在用户控制的方向上，将一些选定的3D曲线、CONS、元素边缘、线元素或六边形边缘投影到一些选定的Faces、WPLANEs和CROSS_SECTIONs上。

CONS或3d曲线的选择可以通过“角”、“PID邻域”或“路径”来辅助。

2.2.3 Parametrical

按照参数插入硬点

此函数在3d曲线段或CONS段上参数定义新的Node或Hot Point。输入参数值可以在0到1之间变化，描述3d曲线或CONS长度的百分比，也可以是距离的精确值加上前缀(~)。

2.2.4 Delete

删除硬点

2.2.5 Muit.Project

批量投影硬点

此功能将一些选定的热点，3D点，焊接点等投影到一些选定的con, 3D曲线或面。

2.2.6 Intersect

相交？应该是定义两组面之间相交的硬点

这个函数定义了两组面和/或相互穿透的FE元素之间的交集。如果需要，可以自动应用拓扑。如果两个组之间可以定义一个封闭的卷，则可以应用布尔运算。

2.2.7 Release

将一个共用的硬点释放掉，换成各面交点上的硬点

这个函数释放位于两个或多个面的极限上的热点。

当一个热点被释放时，相邻的面限制的con也被释放，成为单个con(用红色标记)。导致本地拓扑发生变化。

2.2.8 Clean G.

清除Gap

这个函数删除了可见的重叠或未修剪的faces部分。这个函数自动修复了大多数常见的拓扑问题。它消除了:

  1. 倒塌的缺点
  2. 间隙(面之间的小开口)
  3. 裂缝(未粘贴的缺陷)
  4. 三个或更多con(针面)
  5. 此外，它还对所有可见实体执行拓扑

2.2.9 Rm.Overlap

这个函数删除面中可见的重叠或未修剪的部分

2.3 CONS

在拓扑编辑中，“Constraints”指的是对模型中某些部分施加的限制或约束条件。这些约束可以帮助在修改模型时保持某些特定的几何特性或关系，如保持两个面平行或一个点在特定曲线上。

Paste

面型修补：使得两个相邻的两个边连在一起（红色变黄色~）

这个函数粘贴(连接)两个或多个彼此靠近的CONS片段，这些片段通常是不同Faces的边界。一个新的拓扑条件被创造，这些面成为相邻的。

Release

将一个共用的边释放出来，将其变成不相连。

该函数释放自动加入(粘贴)或通过“CONS>Paste”函数加入的CONS。

Fill Hole

补孔，将指定小于一定范围的内的孔给补上。

这个函数填充Faces上的孔。孔可以是任何形状。这种内部开口可以单独选择，如果它们低于用户指定的直径值，也可以在可见的开口中自动选择。有选项自动创建3D曲线，点或连接点的删除孔，以标记其形状和位置，以供参考。

Open Hole

开孔

该函数在面或FE壳单元上铸造孔和管。

Project

投影：对限制或约束条件进行投影

该函数以CONS或3D Curves的形式，在用户控制的方向上，将一些选定的3D曲线、CONS、元素边缘、线元素或六边形边缘投影到一些选定的Faces、WPLANEs和CROSS_SECTIONs上。

CONS或3d曲线的选择可以通过“角”、“PID邻域”或“路径”来辅助。

Break

补全？

该函数通过在不连续的位置插入热点来“补全”选定的CONS。如果所选的CONS没有任何不连续，则该CONS不会被”补全“。使用转角功能也可以方便地选择CONS。

Cut

使模型对称后进行操作

Intersect

对面型之间的交集进行布尔运算。

这个函数 定义了两组面和/或相互穿透的FE元素之间的交集。如果需要，可以自动应用拓扑。如果两个组之间可以定义一个封闭的卷，则可以应用布尔运算。

Extend

面型衍生（拓展）

#### Extend [Target]

该函数通过指定面作为源面的扩展范围来扩展源面，根据指定的距离或直到它们到达目标面孔或目标面孔组。新面的表面是源表面的扩展。如果需要，可以自动应用拓扑。

Orient

面的法向调整

select

这个函数反转所选面的方向。将要生成的网格元素或已经存在的网格元素也获得统一的方向。

Shrink

此函数向内收缩选定的Face(多个)给定的距离值。

Project

面投影

这个函数将一组面的所有CONS投影到另一组面，也应用拓扑。

Link

子父特征

这个函数定义了一个链接面到一个选定的面，这样两个面将产生兼容的网格。因此，每个应用于一个面的操作，都会自动应用于另一个面。这些操作可以是插入一个热点，切割面等。链接面的位置可以通过三种方式确定：

  • 通过厚度因子
  • 通过用户指定的距离
  • 通过厚度和指定距离的组合

Surface

Surfaces: 这与”Faces”类似，通常指模型中的曲面或平面区域。在拓扑编辑中，处理表面可能包括平滑、调整或重新定义表面的边界。

Coons

空丝曲面，补面

这个函数定义了一个新表面，使用Coons插值原理。这是一个简单的方法来定义一个新的表面由其边界曲线。四边形和三角形参数多patch表面可以被定义。

在数学定义完成后，计算表面作为一个网出现。有能力旋转，平移和缩放视图，以检查其形态。一个相对窗口提供接受或拒绝计算表面的能力。当只有三个边界边被选中时，有能力选择第四个生成的边之间的两个替代定义，一个是直的，另一个是具有类似于其对面的形状。

Fit

根据边界条件生成适应的面

这个函数可以通过修改(重新塑造)一个现有的面来创建一个新的面，这样它就适合选定的边或点实体。边指的是曲线、CONS甚至壳元边，而点实体可以是3D Points、Hot Points或FE-Model节点。

Extend

面延伸

这个函数创建一个新的表面，可以选择一个面，作为现有表面的切向延伸，同时替换原来的表面。延伸长度可以通过抽取和拖动曲面的边缘以数字或图形方式确定。从预览中选择扩展方向。

Volume

创建体模型

Extrude

创建由在路径上移动的曲线轨迹定义的面

Break

这个函数根据它们的表面特征来分割面。

它可以用于具有恒定或可变曲率和曲率不连续曲面的面，以获得更简单的几何形状的面，从而促进以后更好的网格。

Fillet

定义一个圆角或倒角表面

Plane

这个函数可以用来定义一个平面。

为了定义一个新的平面，必须指定一个活动工作平面。此工作平面可以通过选择现有工作平面或通过定义临时工作平面来定义。

Resize

此函数用于沿其表面移动一个面或基于现有表面创建一个新面。

Curves

Curves: 在模型中，“Curves”指在空间中独立存在、不依附网格的线。它的存在就是起到一个辅助的作用，用来清理几何。

Curves是3D空间中的曲线。3D曲线与将要进行网格划分的面也不产生连接。但是，3D曲线对于创建或修改现有几何形状非常有用，因为它们可用于从线框模型创建表面模型，或者通常用于校正或修改表面几何形状。

Transform

这个函数定义新的3D曲线，作为一系列连续选择的3D曲线、CONS或截面片段的变换。变换可以同时平移、旋转和缩放。

Point

Point在ANSA中表示参考点，它不与模型产生连接，在ANSA中常用作空间定位坐标点。

3.1 模型修复

3.1.1 模型内外法向调整

导入模型后，每个面的颜色会有两种情况，黄色（负向）和灰白色（正向）。

需要保证每个大面的整体法向一致。

想要调整方向， 在Topo界面的Faces类别下有Orient选项，选择其下的select选项 将需要调整的表面进行选择，按鼠标中间确定即可调整方向。

Ps：在选择时，尽量使用框选选择。

3.1.2 使边界边顺滑

在Topo界面下 TOPO>Auxiliaries>Fine，这个功能可增加或减少所选3d曲线、CONS段或Face的显示分辨率，言外之意就是将边界线or面型变得更加顺滑。

3.1.4 边界修整

两面边界为红色时表示两面未连接，为黄色表示两面已经共用一条边（已连接）

TOPO>Faces>Topo

该功能可在选定的相邻CONS之间自动创建拓扑条件。

通常情况下，在导入三维模型后将三维模型全选后选择Topo自动修复。然后再将不能修复的边界重新一一修复。

3.1.5 面型切割

Face Cut

TOPO>Faces>Cut

如果应用反向操作(鼠标右键)，此函数将切割(拆分)现有的Face或连接Faces。

它不需要热点的存在，因为它可以直接应用于CONS的任意位置。

投影切割

TOPO>Faces>Proj.Cut

它将一个面分割为一个选定的hot point和它在选定的CONS上的投影。意思就是将一个完整的边界线投影在一个面上，然后把在面上的投影按照此边界线进行切割，切割后为一组hot point。

面型删除

TOPO>Faces>Delete

这个函数删除选定的面。当Face被删除时，邻近的拓扑条件会发生变化。如果要删除的选中的面作为其他链接面的父面，则需要进行额外的删除确认，并提供将所有链接面的转换为几何形状的选项。
